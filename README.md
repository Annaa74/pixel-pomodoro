 

# 🌸 Pixel Pomodoro

A lightweight, playful **Pomodoro Timer** built with **HTML, CSS, and JavaScript**, designed to make productivity fun with **pixel-art GIFs** and **retro sound effects**. Perfect for students, developers, or anyone who wants to stay focused while enjoying a touch of nostalgia.

---

## ✨ Features
- ⏱️ **Pomodoro Technique**: Focus and break cycles to boost productivity.  
- 🎨 **Pixel Aesthetic**: Animated GIFs and pixel icons for a retro vibe.  
- 🔊 **Sound Effects**: Audio cues to signal session changes.  
- 🎮 **Interactive UI**: Dropdown menu to choose your favorite GIF theme.  
- 🌐 **Responsive Design**: Works smoothly across devices and browsers.

## ⚡ Getting Started
1. Clone the repository:
   ```bash
   git clone https://github.com/Annaa74/pixel-pomodoro.git
   ```
2. Open `index.html` in your browser.  
3. Select a GIF theme from the dropdown and start your focus session!
